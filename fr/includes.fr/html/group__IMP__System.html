Description détaillée

Module de contrôle du système ， comprendre IMP Fonctions de base et fonctions associées liées entre les modules.

1 Concepts associés

Le contrôle du système réalise principalement la connexion de divers modules ， Définissez la fonction du flux de données.

1.1 Device

Device C'est une collection pour compléter une certaine fonction (de classe). FrameSource Terminer la sortie des données de la source vidéo ，Encoder Remplissez la fonction de codage vidéo ou de codage d'image. FrameSource avec Encoder Seulement Device le concept de.
Device Juste un concept de collection ， Ce n'est pas un n œ ud de flux de données spécifique.

1.2 Group

Group C'est la plus petite unité d'entrée de données. Device Il peut y avoir plusieurs Group， Chaque Group Une seule entrée de données peut être reçue. Group Peut avoir plusieurs sorties ( 1.3 Output ).
Group C'est aussi un conteneur pour des « fonctions » spécifiques ， Tu peux voir 1.5 Channel Une partie de l'explication.

1.3 Output

Output Est un Group La plus petite unité de sortie de données pour un canal. Group Il peut y avoir plusieurs Output， Chaque Output Un seul canal de données peut être émis.

1.4 Cell

Cell Se réfère à contient Device [[[ Group [[[ Output Collecte d'informations. IMPCell La structure des données est présentée.
Cell Principalement utilisé Bind ( 2 Liaison de module ( Bind ) ). Device [[[ Group [[[ Output Définition ，Output En tant que n œ ud de sortie de données ， et Group En tant que n œ ud d'entrée de données.
dans Bind Noeud de sortie des données de temps Cell Index à la sortie Output， N œ ud de saisie de données Cell Index à saisir Group (Par conséquent, en tant qu'entrée de données Cell，Output Est une valeur dénuée de sens).

1.5 Channel

Channel Fait généralement référence à une seule unité fonctionnelle ，Channel dans Create Temps ( Instancier ) Lorsque la fonction spécifique est attribuée.
Par exemple:

  • pour Encoder， Une Channel Complétez le chemin H264 Code ou JPEG Fonction de codage ， Fonction de codage spécifique ( Types de ， paramètre ) Spécifié lors de la création du canal
  • pour IVS， Une Channel Compléter la fonction d'un algorithme spécifique ， Les paramètres de type d'algorithme spécifiques sont spécifiés lors de la création du canal
  • pour OSD， Avoir le même Channel Concepts similaires Region， Une Region Est une zone de superposition spécifique ， Peut être PIC( image )，COVER( Occlure ) Attendre
  • pour FrameSource， Une Channel Sortie d'une image originale ，FrameSource de Channel Est en fait Group

Channel En tant qu'unité fonctionnelle ， ont généralement besoin Register À Group dans (FrameSource sauf )， Pour recevoir les données. Channel Inscrivez-vous à Group Milieu et arrière ， Aura Group Données saisies.
différent Device de Group peut Register de Channel Les chiffres sont également différents.
