Description détaillée

Module OSD, peut superposer des images, des bitmaps, des lignes droites et des bo je tes rectangulaires sur le flux vidéo.

de l'audio_summary 1

OSD Le nom complet est On-Screen Display .

2 concepts associés

osd_region 2.1

Region Est-ce que la zone superposée ， dans API Abréviation chinoise Rgn . Region Avoir certaines informations sur l'image ， Peut passer OSD Une fois les modules superposés ， Combiné avec l'image d'arrière-plan dans une image. Pour la superposition d'image ， Peut également être réalisé Alpha effet. 2.1 Type de région .

2.1 Type de région

Region Il existe plusieurs types ， Elles sont:
OSD_REG_LINE :ligne droite
OSD_REG_RECT :Rectangle
OSD_REG_BITMAP : bitmap
OSD_REG_COVER : Occlusion
OSD_REG_PIC :image
parmi eux ， La différence entre le bitmap et l'image est ， Bitmap n'effectue qu'une couverture monochrome des pixels ， Et l'image est RGBA Image Alpha Recouvrir.

du module osd_fun 3

OSD Le module prend en charge les lignes, les bo je tes rectangulaires, la superposition bitmap, l'occlusion rectangulaire et la superposition d'image. Les lignes, les bo je tes rectangulaires et les bitmaps sont implémentés par logiciel; l'occlusion rectangulaire et la superposition d'image sont implémentées par le matériel.

du module osd_use 4

OSD L'utilisation de comporte généralement les étapes suivantes

  1. créer OSD grouper
  2. Lier OSD Regrouper dans le système
  3. créer OSD surface
  4. inscrit OSD Zone à OSD Dans le groupe
  5. Installation OSD Attributs de zone de groupe et attributs de zone
  6. Installation OSD Commutateur de fonction
