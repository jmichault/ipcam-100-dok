Detailed Description

OSD Module ， Can superimpose pictures, bitmaps, straight lines, rectangular boxes on the video stream.

1 Module introduction

OSD The full name is On-Screen Display . The function of the module is to superimpose information such as lines and pictures in each film source.

2 Related concepts

2.1 Region

Region Is the superimposed area ， in API Chinese abbreviation Rgn . Each Region Have certain image information ， Can pass OSD After the modules are superimposed ， Combined with the background image into a picture. For picture overlay ， Can also be achieved Alpha effect. For detailed introduction of various overlay types, please refer to 2.1 Region type .

2.1 Region type

Region There are several types ， They are:
OSD_REG_LINE :straight line
OSD_REG_RECT :Rectangle
OSD_REG_BITMAP :bitmap
OSD_REG_COVER : Occlusion
OSD_REG_PIC :image
among them ， The difference between bitmap and picture is ， Bitmap only performs monochrome coverage of pixels ， And the picture is RGBA Image Alpha Overlay.

3 Module function

OSD The module supports lines, rectangular boxes, bitmap overlay, rectangular occlusion, and image overlay. Lines, rectangular boxes and bitmaps are implemented by software; rectangular occlusion and image overlays are implemented by hardware.

4 Module usage

OSD The use of generally has the following steps

  1. create OSD group
  2. Bind OSD Group into the system
  3. Group into the system OSD area
  4. registered OSD Area to OSD In the group
  5. Set up OSD Group area attributes and area attributes
  6. Group area attributes and area attributes OSD Function switch
