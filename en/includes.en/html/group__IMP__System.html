Detailed Description

System control module ， include IMP Basic functions and related functions bound between modules.

1 Related concepts

System control mainly realizes the connection of various modules ， Define the function of the data flow. The following are some important concepts:

1.1 Device

Device It is a collection to complete a certain (class) function. Such as FrameSource Complete the output of video source data ，Encoder Complete the function of video coding or image coding. here FrameSource with Encoder Just Device the concept of.
Device Just a collection concept ， It is not a specific data flow node.

1.2 Group

Group It is the smallest unit of data input. One Device There can be multiple Group， Each Group Only one data input can be received. Group Can have multiple outputs ( 1.3 Output ).
Group It's also a container for specific "functions" ， You can see 1.5 Channel Part of the explanation.

1.3 Output

Output Is an Group The smallest unit of data output for one channel. One Group The smallest unit of data output for one channel. One Output， The smallest unit of data output for one channel. One Output Only one channel of data can be output.

1.4 Cell

Cell Refers to contains Device [[[ Group [[[ Output Collection of information. To IMPCell The data structure is presented.
Cell Mainly used Bind ( 2 Module binding ( Bind ) ). according to Device ). according to Group ). according to Output Definition ，Output As data output node ， and Group As the data input node.
in Bind Time data output node Cell Index to output Output， Data entry node Cell Index to input Group (Therefore as data input Cell，Output Is a meaningless value).

1.5 Channel

Channel Usually refers to a single function unit ，Channel Usually refers to a single function unit Create Time ( Instantiate ) When the specific function is assigned.
E.g:
